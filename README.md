 ### `正则表达式由两部分组成`
 - 元字符
 - 修饰符
 ## 常用的元字符 
 ```
 1.量词元字符，设置出现的次数
 * 零到多次
 + 一到多次
 ？ 零或一次
 {n} 出现n次
 {n,} 出现n到多次
 {n,m} 出现n到m次
 ```
 ```
 2.特殊元字符:单个或者组合在一起代表特殊的含义
 \ 转义字符
 . 除\n(换行符)以外的任意字符
 ```
